Drowsy Driver Detection

The Drowsy Driver Detection system is an advanced deep learning-based project that aims to enhance road safety by detecting driver fatigue. Utilizing CNN, VGG16, VGG19, AlexNet, and ResNet9 models, the system accurately classifies driver states into 'drowsy' and 'non-drowsy'. The model processes real-time video frames to identify signs of fatigue, such as eye closure and yawning, ensuring timely alerts to prevent accidents.

This project involves extensive dataset augmentation and preprocessing to achieve high accuracy and robustness. The system is designed for real-world applications in automotive safety, integrating seamlessly with existing vehicle monitoring systems.

Technologies Used: Deep Learning, Computer Vision, Image Processing, Real-time Monitoring
Libraries & Frameworks: TensorFlow, Keras, OpenCV, NumPy, Matplotlib
Languages: Python
Tools: Jupyter Notebook, Google Colab
