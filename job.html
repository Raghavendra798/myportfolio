Job Recommendation System

An intelligent job recommendation system designed to match job seekers with the most relevant job opportunities based on their skills, experience, and preferences. The system utilizes **machine learning and data-driven approaches** to enhance job discovery and career growth.

The recommendation engine analyzes job descriptions and candidate profiles to generate personalized job suggestions. It integrates **skill matching, keyword extraction, and ranking algorithms** to deliver high-accuracy recommendations.

With **real-time filtering, an intuitive UI, and API integration**, the system ensures seamless job searching and application tracking.

Key Features:

  • Personalized job recommendations based on skills and experience
  • AI-driven matching using machine learning algorithms
  • Real-time job posting updates and filtering
  • Interactive user dashboard with saved job options
  • Integration with LinkedIn and external job boards
  • Resume parsing and keyword-based job ranking

Technologies Used: Streamlit, Pandas, Scikit-learn

Frontend: Html,Css,Js

database: Postgrey sql

Backend: Python,Flask
