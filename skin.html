Skin Cancer Prediction System

A deep learning-based multi-class classification model designed to predict different types of skin cancer using medical imaging. The system utilizes **Convolutional Neural Networks (CNNs)** to identify and classify cancerous lesions with high accuracy.

The model is trained on the **ISIC skin cancer dataset**, which contains diverse oncological images. Various image augmentation techniques and **transfer learning** with models like **SqueezeNet, ResNet, and VGG16** enhance accuracy.

The system helps dermatologists and medical professionals in early cancer detection, improving patient outcomes with automated diagnosis.

Key Features:

  • Multi-class classification of skin cancer types
  • Deep learning-based feature extraction
  • Augmented dataset with 2500 images per class
  • Transfer learning using SqueezeNet and ResNet
  • Web-based UI for easy image upload and classification
  • Accuracy, precision, and recall evaluation metrics
